git

<コマンド>
ファイルの選択　(git add ファイル名)
ファイルの記録　(git commit -m "メッセージ")
リモートに登録　(git remote add リモート名　リモートのurl)
リモートにアップロード　(git push origin//リモート名  master)
リモートから保存　(git pull origin//リモート名　master)
変更の確認 (git status),変更内容の確認(git diff)
コミット履歴確認　(git log)

Command Line
<コマンド>
ファイルの作成　(touch ファイル名)
ファイルの中身の表示(cat ファイル名)
ディテクトリ(フォルダ)の作成　(mkdir ディレクトリ名)
ディレクトリの移動　(cd ディレクトリ名)
カレントディレクトリの確認　(pwd)
ディレクトリの中身を表示　(ls)
一つ親のディレクトリに移動　( cd .. )
ファイルを移動させるコマンド　(mv ファイル名　ディレクトリ名)
　↑　ファイル名の変更も可能　
ファイルのコピー　(cp コピーするファイル名　新しいファイル名)
ファイルの削除　( rm ファイル名 ) ディレクトリの場合　-r をつける


自分が今いるディレクトリ (カレントディレクトリ)
一番親のディレクトリ　(ルートディレクトリ)
作業するときの起点のディレクトリ　(ホームディレクトリ)



現在のディレクトリの場所を確認する	pwd	dir
ファイルやディレクトリの情報を表示する	ls -al	dir
ディレクトリ間の移動	cd	cd (or) chdir
ディレクトリを作成	mkdir	md (or) mkdir
ディレクトリを移動	mv -r	move
ディレクトリをコピー	cp -r	xcopy /e /c /h
ディレクトリを削除	rm -r	del (or) erase
(or) rd /s (or) rmdir /s
ファイルを移動	mv	move
ファイルをコピー	cp	copy
ファイルを削除	rm	del (or) erase
テキストファイルの内容を表示	cat (or) less	type
